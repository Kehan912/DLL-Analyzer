# DLL 分析工具

一款基于 Python 和 Tkinter 的 DLL 文件分析工具，支持拖入或选择 DLL 文件，查看其基本信息、导出函数以及节信息。界面简洁，操作方便，适合开发者或安全研究者快速分析 DLL 文件。

---

## 功能

- 拖入 DLL 文件或通过文件选择器加载 DLL。
- 显示 DLL 的基本信息：
  - 文件名、路径、大小
  - PE 架构（32/64 位）
  - 编译时间
  - 节数量
- 显示 DLL 的导出函数列表：
  - 支持搜索函数名
  - 支持复制选中或全部函数名
- 显示节信息：
  - 节名称、虚拟地址、虚拟大小、原始大小
- 界面简洁易用，中文显示。
